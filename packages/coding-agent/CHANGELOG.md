## [0.1.1] - 2026-02-24

### Added

- Persistent memory system (procedural, episodic, semantic) stored per-project in `.pi/memory/`
- Memory tools: `memory_write`, `memory_read`, `memory_update`, `memory_delete`
- `/memory` command with list, show, search, delete, compact subcommands
- Timestamp injection on all user messages for time-awareness
- Memory toggle in `/settings` (on by default)
